## 📝 使用说明

### v2.1新特性：简化时间输入

**时间格式：**
- 简化格式：`7-1 1000` 表示 `2026年7月1日10:00`
- 年份默认为当前年份（2026年）

**输入示例：**
- `7-1 1000` → 2026年7月1日10:00
- `12-25 1530` → 2026年12月25日15:30
- `1-5 0800` → 2026年1月5日08:00

### v2.1新特性：潮高自动计算

系统会自动从潮水表中查询潮高数据：
- 靠泊前1小时硇洲潮高（B）
- 靠泊时湛江港潮高（C）
- 离泊时湛江潮高（F）
- 离泊后1小时硇洲潮高（G）

无需手动输入这些参数！

## 使用示例

### 示例1：生成601泊位文档（含潮水计算）

1. 打开 http://localhost:8000
2. 填写基本信息：
   - 中文船名：`测试船舶`
   - 英文船名：`TEST SHIP`
   - 到港时间：`2026-10-22 14:00`
   - 靠泊时间：`2026-10-22 16:00`
   - 卸货量：`50000`
   - 整船货量：`50000`
   - 船长：`225`
   - 船宽：`32.2`
   - 载重吨：`60000`
   - 总吨：`35000`
   - 净吨：`22000`
   - 泊位类型：`601泊位`
   - 代理公司：`中国湛江外轮代理有限公司`

3. 填写潮水计算参数：
   - 航道水深（Y）：`18`
   - 靠泊时间（A）：`2026-10-22 11:30`
   - 靠泊前1小时硇洲潮高（B）：`2.7`
   - 靠泊时湛江港潮高（C）：`3.3`
   - 船舶靠泊吃水（D）：`18.47`
   - 离泊时间（E）：`2026-10-23 18:00`
   - 离泊时湛江潮高（F）：`1.6`
   - 离泊时已1小时后的硇洲潮高（G）：`1.77`
   - 船舶离泊吃水（H）：`13`
   - 龙骨高度（I）：`50.51`

4. 点击"生成文档"按钮
5. 等待生成完成
6. 在历史记录中下载生成的文档

### 潮水计算说明

系统会自动根据选择的泊位类型和输入的潮水参数，计算：
- **靠泊前1小时**：系统自动计算A时间之前的1小时
- **离泊后1小时**：系统自动计算E时间之后的1小时
- **富余水深**：根据不同泊位类型的公式自动计算
- **通航富余高度**：专门针对300泊位的海湾大桥通航计算

### 泊位类型与航道水深对应关系

## 常见问题

### Q: 端口8000被占用怎么办？

修改 `app.py` 最后一行：
```python
uvicorn.run(app, host="0.0.0.0", port=8080)  # 改为其他端口
```

### Q: 生成的文档在哪里？

生成的文档保存在 `output` 目录中。

### Q: 如何清空历史记录？

删除 `maritime_documents.db` 文件：
```bash
rm maritime_documents.db
```

### Q: 如何备份数据？

备份以下文件：
```bash
cp maritime_documents.db maritime_documents.db.backup
cp -r output output.backup
```
